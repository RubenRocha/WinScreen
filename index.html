WinScreen by RubenRocha

Something like "Archey" for Windows.

I made this because I saw some guys over at /g/ making their own versions in batch, python, vb, C# so i decided and since everyone was like GO MAKE IT IN C++. I decided to make it.

Dependencies

None, just a Windows(NT+) Installation.

Install

Download it and then run it from the cmd or just run the Monitor batch file.
